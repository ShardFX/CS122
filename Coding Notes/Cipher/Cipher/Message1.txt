This is a very rudimentary RSGallery2 CB Plugin. I created it for an immediate need and plan to expand it to be more robust as part of my thanks to the community. It currently only contains a list of galleries for the user with the gallery name and a thumbnail of a random image in the gallery. If the user is in his or her own gallery, a link to edit the users gallery also displays.
